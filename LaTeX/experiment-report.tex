\begin{abstract}
本文详细报告OraSRS（Oracle Security Root Service）协议的实验评估结果。
OraSRS是一种创新的去中心化威胁情报协议，采用咨询式服务模式，提供风险评分和建议，
由客户端自主决策是否采取行动。本实验报告评估了OraSRS协议在性能、隐私保护、
系统鲁棒性和可扩展性方面的表现。实验结果表明，OraSRS协议在保护隐私的同时，
实现了高效的风险评估和威胁检测，验证了所有研究假设。
\end{abstract}
